🚀 Deploy & Setup Guide

Go live in under 15 minutes — zero cost.

Step 1 — Host on Vercel (5 min)
  1. Go to github.com → New repository → name it increation-survey
  2. Upload this HTML file → click Commit
  3. Go to vercel.com → Sign up free → Import from GitHub
  4. Select the repo → Deploy → you get a URL like increation-survey.vercel.app
  5. Optional: connect survey.increation.ae in Vercel settings
Step 2 — Wire Google Sheets (10 min)
  1. New Google Sheet → name it Increation Surveys
  2. Row 1 headers: ID · Project · Client · Location · Surveyor · Date · Status · Submitted · Type · GFA · Snags · High Snags · Summary
  3. Extensions → Apps Script → paste this code → Save → Deploy as Web App (Execute as: Me, Access: Anyone)
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var d = JSON.parse(e.postData.contents);
  sheet.appendRow([d.id,d.project,d.client,d.location,d.surveyor,
    d.date,d.status,d.submitted_at,d.survey_type,d.gfa_sqm,
    d.snag_count,d.high_snags,d.summary]);
  return ContentService.createTextOutput('ok');
}
function doGet() { return ContentService.createTextOutput('running'); }
  1. Copy the Web App URL after deploying
  2. Open this HTML file in a text editor → find SHEETS_URL = 'YOUR_GOOGLE' → replace with your URL → save → re-upload to GitHub
